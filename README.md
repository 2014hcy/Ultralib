# Ultralib - a Ultrasonic sensor's lib. 

__It's__ can __helps__ the __*beginner*__ using __Ultrasonic sensor__ very __easy__.

## Install

1. __Install__ the __[Arduino IDE](https://github.com/arduino/arduino-ide/releases)__ or __[Arduino CLI](https://github.com/arduino/arduino-cli/releases)__.
2. __Install__ __[Ultralib](https://github.com/2014hcy/Ultralib/archive/refs/heads/main.zip)__.
3. __Extract__ zip file to __your folder__.
4. __Use__ my __*easy_WiFi*__ lib to makes a __perfect works__!

## Instructions

__*"Ultra ultra(trigPin, echoPin)"*__ is to __creat an object__.<br>
__*"ultra.read()"*__ can __return the distance__.
